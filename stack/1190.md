
#### 题解
* 1.利用stack 来递归，层层递进

```java
class Solution {
    public String reverseParentheses(String s) {
        Deque<String> stack = new LinkedList();
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);
            if (ch == '(') {
                stack.push(sb.toString());
                // sb 归零
                sb.setLength(0);
            }
            else if (ch == ')') {
                sb.reverse();
                sb.insert(0, stack.pop());

            }
            else {
                sb.append(ch);
            }
        }
        return sb.toString();


    }
}
```
