#### 寻找排序数组中的最小值




```java
class Solution {
    public int findMin(int[] nums) {
        int left = 0;
        int right = nums.length-1;
        while (left < right) {
            int mid = (left + right) >>> 1;
            if (nums[mid] > nums[right]) {
                left = mid + 1;
            }
            else if(nums[mid] < nums[right]) {
                right = mid;
            }
            else right -= 1;
            // 数组要是相同的话，right进行处理
        }
        return nums[left];

    }
}
```
