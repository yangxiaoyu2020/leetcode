##### 官方的解法
```
其实就是暴力解法

我就把这种解法叫做圈地盘吧

前缀和 的解法

S1 - S2 = S(1->2)
求最大值S(1->2)
所以 官方用了treeSET 来进行排序
取最大值 S1 - k(这里的k就是S(1-2)) 用了 tree 找 S2 用了ceiling 所以找到了最接近的 最大值 向上取整（这么理解）
最后 找最大的 S(1-2)

一般这种区间最大最小的问题，不是dp 就是 前缀和 的解法
```

```java
class Solution {
    public int maxSumSubmatrix(int[][] matrix, int k) {
        int m = matrix.length;
        int n = matrix[0].length;
        int ans = Integer.MIN_VALUE;
        for (int i = 0; i < m; i++) {
            int[] sum = new int[n];

            for (int j = i; j < m; j++) {
                for (int k1 = 0; k1 < n; k1++) {
                    sum[k1] += matrix[j][k1];
                }
                TreeSet<Integer> set = new TreeSet();
                set.add(0);
                int s = 0;
                for (int v: sum) {
                    s += v;
                    Integer ceiling = set.ceiling(s - k);
                    if (ceiling != null) {
                        ans = Math.max(ans, s-ceiling);
                    }
                    set.add(s);
                }

            }

        }
        return ans;

    }
}
```

##### 优化的做法



```java
class Solution {
    public int maxSumSubmatrix(int[][] matrix, int k) {
        int m = matrix.length;
        int n = matrix[0].length;
        int maxTmp = Integer.MIN_VALUE;
        int sum = 0;

        for (int i = 0; i < m; i++) {
            int[] b = new int[n];
            for (int j = i; j < m; j++) {
                sum = 0;
                for (int l = 0; l < n; l++) {
                    b[l] += matrix[j][l];
                }
                maxTmp = Math.max(maxTmp, dpmax(b, k));
                if (maxTmp == k) return k;
            }
        }
        return maxTmp;
    }

    private int dpmax(int[] arr, int k) {
        int rollSum = arr[0], rollMax = rollSum;
        // O(rows)
        for (int i = 1; i < arr.length; i++) {
            if (rollSum > 0) rollSum += arr[i];
            else rollSum = arr[i];
            if (rollSum > rollMax) rollMax = rollSum;
        }
        if (rollMax <= k) return rollMax;
        // O(rows ^ 2)
        int max = Integer.MIN_VALUE;
        for (int l = 0; l < arr.length; l++) {
            int sum = 0;
            for (int r = l; r < arr.length; r++) {
                sum += arr[r];
                if (sum > max && sum <= k) max = sum;
                if (max == k) return k; // 尽量提前
            }
        }
        return max;
    }
}
```
