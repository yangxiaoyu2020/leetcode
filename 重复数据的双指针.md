#### 解题思路
```txt
java 双指针
[1,1,2,3,4,5,5,6] -> [1, 2,2,3,4,5,5,6] -> [1,2,3,3,4,5,5,6] -> [1,2,3,4,4,5,6] -> 依次类推
实现 重复数据的覆盖。
```

##### 代码
```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int n = nums.length;
        if (n <= 2) return n;
        int slow = 2;
        int fast = 2;
        while (fast < n) {
            if (nums[slow-2] != nums[fast]) {
                nums[slow] = nums[fast];
                slow++;
            }
            fast++;
        }
        return slow;

    }
}
```

快慢指针
**这个是去除指定的重复数据**

```java
public int removeDuplicates(int[] nums) {
    if (nums.length == 0) return 0;
    int i = 0;
    for (int j = 1; j < nums.length; j++) {
        if (nums[j] != nums[i]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}
```
