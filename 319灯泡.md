#### 解题思路
计算完全平方数，
若一个数能表示成某个整数的平方的形式，则称这个数为完全平方数
完全平方数 的 因数为奇数个

### 代码
```java
class Solution {
    public int bulbSwitch(int n) {
        int res = 1;
        while (res*res <= n) {
            res += 1;
        }
        return res-1;

    }
}
```
用粗俗的方法表示下完全平方数

```java
class Solution {
    public int bulbSwitch(int n) {
        if (n <= 0) return 0;
        int dp = 1;
        for (int i = 2; i <= n; i++) {
            int count = (int)Math.sqrt(i);
            if (count * count != i) {
                dp = dp;
            }
            else dp = dp + 1;
        }
        return dp;
    }


    private int help(int i ) {
        int res = 0;
        if (i <= 4) return 1;
        for (int j = 2; j < i/2; j++) {
            if (i % j == 0) {
                if (i /j == j) {
                    res += 1;
                }
                else res += 2;

            }
        }
        return res;
    }
}
```
