那么当前我就能获胜，我不用擦黑板，否则别人获胜。

```java
class Solution {
    // 这两个混蛋一天不上班，就在这里玩游戏，你说气人不气人

    public boolean xorGame(int[] nums) {
        int n = nums.length;
        if (n % 2 == 0) return true;
        int res = 0;
        for (int x : nums) {
            res ^= x;
        }
        return res == 0;

    }
}
```
